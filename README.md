# 🚀 Java OOP Projects

This repository contains two simple Java programs built with Java to demonstrate real-world applications of Object-Oriented Programming.

## 📂 Projects

### 1️⃣ Car Rental System
A program that simulates a basic car rental service.  
It allows users to:
- 🚗 Add cars to the system
- 📅 Rent a car for a number of days
- 🔄 Return a car
- 💰 Calculate rental costs
- 👥 Manage customers and rental records  

### 2️⃣ Employee Payroll System
A program that models a simple payroll management system.  
It allows users to:
- 👨‍💼 Add employees (full-time and part-time)
- 💵 Calculate salaries based on type of employee
- 🗂️ Manage employees in a payroll list
- 📝 Display employee details with salary information
- ❌ Remove employees by ID
